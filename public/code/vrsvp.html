VRSVP

⇐ Go back

This is a Discord bot I have created to help people sign up for events. Unlike similar bots, most content can be changed by a administrator after creation. It has the ability to set limits on signups per time slot or person.

The bot is currently in a limited beta test rollout. Contact me for adding the bot to your server. It can also be self-hosted using the source code.

Examples

These show what a generated RSVP form looks like, the buttons someone can use to sign up, and the command used to generate the shown form.

Command Options

  • start: HammerTime timestamp
  • slots: Timestamp slot count
  • duration: Timestamp slot length in minutes
  • rsvp-limit-per-person: Optional, a person cannot sign up for more slots than this
  • rsvp-limit-per-slot: Optional, a slot cannot have more people signed up than this

Editing

The event description can be edited from within Discord by a server administrator. This allows correcting timestamps, giving more information, re-arranging individuals, and more. It is helpful to be aware of Discord text formatting.

  • These lines in event description set RSVP limits:
    • Maximum number of people that can RSVP for a single slot: (limit)
    • Maximum number of slots a person can RSVP for: (limit)
  • Any line that starts with this is a slot:
    • > #

Discord Limitations

  • Event description editing is limited to 4,000 characters, so long announcements are best done outside the poll
  • Many people using this at once might rate-limit the bot, always double-check your RSVP went through
  • We cannot have more than 25 timestamp slots

Future Features

  • Optimization to avoid potential rate limits from Discord
  • Disabling slot toggle buttons if limits are reached
  • Signup queue for slots in case anyone drops out
  • Event description color code
  • Outputting CSV text for record keeping
  • User role manipulation upon signup
  • Notification/reminder messages
  • Event templates
