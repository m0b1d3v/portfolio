This is a Discord bot I have created to help people sign up for events. Unlike similar bots, most content can be changed by a administrator after creation. It has the ability to set limits on signups per time slot or person.

The bot is currently in a limited beta test rollout. Contact me for adding the bot to your server. It can also be self-hosted using the source code.

Example

This (small) screenshot shows what a generated RSVP form looks like, the buttons someone can use to sign up, and the command used to generate the shown form.

Command Options

  • start: HammerTime timestamp.
  • blocks: Number of slot groups to build.
  • slots: Number of slots per block to build.
  • duration: How long each slot lasts in minutes.
  • rsvp-limit-per-person: Optional, a person cannot sign up for more slots than this.
  • rsvp-limit-per-slot: Optional, a slot cannot have more people signed up than this.

Editing

The event description and each block can be edited from within Discord by a server administrator. This allows correcting timestamps, giving more information, re-arranging individuals, and more. It is helpful to be aware of Discord text formatting.

  • These lines in event description set RSVP limits:
    • Maximum number of people that can RSVP for a single slot: (limit)
    • Maximum number of slots a person can RSVP for: (limit)
  • Any block line that starts with this is a slot:
    • > #

Discord Limitations

  • RSVP form cannot be more than 6,000 characters so long announcements are best done before or after the poll
  • We cannot have more than 10 blocks so long events are best broken up into many polls
  • We cannot have more than 25 signup buttons (slots)
  • Many people using this at once might rate-limit the bot, always double-check your RSVP went through

Future Features
